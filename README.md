# Big Data Homework

Homework assignments for the *Big Data: From Theory to Practice* course.

## Contents

| Folder | Topic | Dataset |
| ------ | ----- | ------- |
| `data-pre-processing` | Data cleaning, encoding, scaling, and exploratory visualisation | Titanic |
| `python-classification` | Supervised classification with Random Forest | Iris |
| `clustering` | Unsupervised clustering — K-Means, DBSCAN, Agglomerative | Iris |
| `neural-network` | Neural network models — Perceptron, Shallow MLP, Deep MLP | Iris |

Each folder contains a Jupyter notebook (`.ipynb`) that runs end-to-end, a written report (`report.md`), and the figures referenced in the report under `figures/`.

## Environment

Python 3.x with the following packages:

```
numpy
pandas
matplotlib
seaborn
scikit-learn
scipy
jupyter
```

Install with:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn scipy jupyter
```
